ProvMonitor
===========

Copyright (c) 2016 Universidade Federal Fluminense (UFF).
All rights reserved.

ProvMonitor is a tool for capturing implicit provenance from scientific workflows.

Team
----

The ProvMonitor team is composed by the following researchers.

* Vanessa Braganholo (UFF)
* Daniel Oliveira (UFF)
* Leonardo Murta (UFF)
* Vitor Neves (UFF)
* Kary Ocaña (LNCC)
* Fabrício Nogueira (UFF)

Publications
------------

* [V. C. Neves, V. Braganholo and L. Murta, "Implicit provenance gathering through configuration management," Software Engineering for Computational Science and Engineering (SE-CSE), 2013 5th International Workshop on, San Francisco, CA, 2013, pp. 92-95.] (http://dx.doi.org/10.1109/SECSE.2013.6615105)
